# 매일 시작 브리핑 워크플로우

// turbo-all

대화를 시작할 때 아래 순서로 진행합니다:

1. task.md를 읽어 현재 진행 상황 파악
```
cat /Users/dyl/.gemini/antigravity/brain/c48e21bb-1744-4bb3-9233-e30f4f35ddfd/task.md
```

2. 오늘 날짜 및 카카오 비즈니스 앱 심사 현황 체크 (심사 신청일: 2026-03-04, 예상 완료: 2026-03-07)
   - 현재 날짜가 2026-03-07 이후이면 → 카카오 심사 결과를 확인했는지 사장님께 물어보기
   - 아직 심사 중이면 → "카카오 비즈니스 앱 심사 중 (D-X일 남음)" 안내

3. 아래 내용을 포함하여 브리핑:
   - ✅ 어제까지 완료된 작업
   - 🔴 오늘 해야 할 작업 (우선순위 순)
   - ⏳ 대기 중인 것 (심사 등)

4. 음성으로 브리핑 시작 알림:
```
say "좋은 아침입니다. 노아도 일일 브리핑을 시작합니다."
```

5. 작업 완료 시 음성 알림:
```
say "작업이 완료되었습니다."
```
